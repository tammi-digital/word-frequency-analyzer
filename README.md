Python script that analyzes a text file and counts word frequency
# Word Frequency Analyzer

This Python program analyzes a text file and counts how many times each word appears.

Example input:

Python is great. Python is fun. I love learning Python!

Example output:

python: 3
is: 2
great: 1
fun: 1
i: 1
love: 1
learning: 1

Skills demonstrated:

- Python file handling
- Text processing
- Collections module
- Regular expressions
